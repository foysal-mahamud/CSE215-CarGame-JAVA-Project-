                                                                                                                                                         ## Step 4: Configure Image Assets
                                                                                                                                                         
                                                                                                                                                         ### Understanding Image Requirements
                                                                                                                                                         
                                                                                                                                                         The CSE215-CarGame requires the following image files in PNG format:
                                                                                                                                                         
                                                                                                                                                         | Filename | Purpose | Recommended Size | Format |
                                                                                                                                                         |----------|---------|-----------------|--------|
                                                                                                                                                         | car1.png | Player vehicle 1 | 40-60 pixels wide | PNG with transparency |
                                                                                                                                                         | car2.png | Player vehicle 2 | 40-60 pixels wide | PNG with transparency |
                                                                                                                                                         | car3.png | Player vehicle 3 | 40-60 pixels wide | PNG with transparency |
                                                                                                                                                         | tree.png | Road obstacle | 40-60 pixels wide | PNG with transparency |
                                                                                                                                                         
                                                                                                                                                         ### How to Add Images to the Project
                                                                                                                                                         
                                                                                                                                                         #### Method 1: Find Images from Reference Repository (Recommended)
                                                                                                                                                         
                                                                                                                                                         1. **Visit Reference Repository:**
                                                                                                                                                         2.    - Go to: https://github.com/return007/car-racing-game/tree/master/images
                                                                                                                                                               -    - This contains sample images compatible with the game
                                                                                                                                                                
                                                                                                                                                                    - 2. **Download Images:**
                                                                                                                                                                      3.    - Right-click on `car_self.png` → Download as `car1.png`
                                                                                                                                                                            -    - Right-click on `car_left_1.png` → Download as `car2.png`
                                                                                                                                                                                 -    - Right-click on `car_left_2.png` → Download as `car3.png`
                                                                                                                                                                                      -    - Right-click on `tree.png` → Download as `tree.png`
                                                                                                                                                                                       
                                                                                                                                                                                           - 3. **Add Images to NetBeans Project:**
                                                                                                                                                                                            
                                                                                                                                                                                             4.    **Step 3a: In NetBeans**
                                                                                                                                                                                             5.   - In the **Projects** panel (left side), expand the project tree
                                                                                                                                                                                                  -    - Right-click on the **`images` folder**
                                                                                                                                                                                                       -    - Select **"Open in Explorer"** (Windows) or **"Show in Finder"** (Mac)
                                                                                                                                                                                                        
                                                                                                                                                                                                            -    **Step 3b: Copy Image Files**
                                                                                                                                                                                                            -       - The file browser opens showing the `images` folder
                                                                                                                                                                                                            -      - Copy the 4 PNG files you downloaded (car1.png, car2.png, car3.png, tree.png)
                                                                                                                                                                                                            -     - Paste them into this `images` folder
                                                                                                                                                                                                        
                                                                                                                                                                                                            -    **Step 3c: Refresh in NetBeans**
                                                                                                                                                                                                            -       - Return to NetBeans
                                                                                                                                                                                                            -      - Right-click on the **`images` folder** in Projects panel
                                                                                                                                                                                                            -     - Select **"Refresh"** (or press **F5**)
                                                                                                                                                                                                            -    - The image files should now appear in the folder
                                                                                                                                                                                                             
                                                                                                                                                                                                                 - #### Method 2: Create Your Own Images
                                                                                                                                                                                                             
                                                                                                                                                                                                                 - 1. **Using an Image Editor:**
                                                                                                                                                                                                                   2.    - Open GIMP (free: https://www.gimp.org/)
                                                                                                                                                                                                                         -    - Or use Photoshop, Paint.NET, etc.
                                                                                                                                                                                                                          
                                                                                                                                                                                                                              - 2. **Create Car Images:**
                                                                                                                                                                                                                                3.    - Create a new 50x40 pixel image
                                                                                                                                                                                                                                      -    - Draw a simple car shape (rectangle with wheels)
                                                                                                                                                                                                                                           -    - Make the background transparent
                                                                                                                                                                                                                                                -    - Export as PNG with name `car1.png`
                                                                                                                                                                                                                                                     -    - Repeat for `car2.png` and `car3.png`
                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                          - 3. **Create Tree Image:**
                                                                                                                                                                                                                                                            4.    - Create a new 40x50 pixel image
                                                                                                                                                                                                                                                                  -    - Draw a simple tree (green triangle on brown rectangle)
                                                                                                                                                                                                                                                                       -    - Make the background transparent
                                                                                                                                                                                                                                                                            -    - Export as PNG with name `tree.png`
                                                                                                                                                                                                                                                                             
                                                                                                                                                                                                                                                                                 - 4. **Add to Project:**
                                                                                                                                                                                                                                                                                   5.    - In NetBeans, right-click **`images` folder**
                                                                                                                                                                                                                                                                                         -    - Select **"Open in Explorer"**
                                                                                                                                                                                                                                                                                              -    - Copy your PNG files into this folder
                                                                                                                                                                                                                                                                                                   -    - Return to NetBeans and press **F5** to refresh
                                                                                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                                                                        - ### Verifying Images in Code
                                                                                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                                                                        - The `CarGame.java` file loads images like this:
                                                                                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                                                                        - ```java
                                                                                                                                                                                                                                                                                                          image1 = ImageIO.read(new File("images/car1.png"));
                                                                                                                                                                                                                                                                                                          image2 = ImageIO.read(new File("images/car2.png"));
                                                                                                                                                                                                                                                                                                          image3 = ImageIO.read(new File("images/car3.png"));
                                                                                                                                                                                                                                                                                                          treeImage = ImageIO.read(new File("images/tree.png"));
                                                                                                                                                                                                                                                                                                          ```
                                                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                          **Important:** The image file paths are relative to where the Java program runs. When running from NetBeans, it looks for images in the `images/` folder at the project root.

                                                                                                                                                                                                                                                                                                          ## Step 5: Build and Run the Game
                                                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                          ### Build the Project
                                                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                          1. **Clean Build (Recommended First Time):**
                                                                                                                                                                                                                                                                                                          2.    - Go to **Build → Clean and Build Project** (or press **Shift+F11**)
                                                                                                                                                                                                                                                                                                                -    - NetBeans will compile all Java files
                                                                                                                                                                                                                                                                                                                     -    - Watch the output in the bottom panel
                                                                                                                                                                                                                                                                                                                          -    - You should see: `"BUILD SUCCESSFUL"`
                                                                                                                                                                                                                                                                                                                           
                                                                                                                                                                                                                                                                                                                               - 2. **If Build Fails:**
                                                                                                                                                                                                                                                                                                                                 3.    - Check the **Output** panel for error messages
                                                                                                                                                                                                                                                                                                                                       -    - Most common issue: Missing image files (see Step 4)
                                                                                                                                                                                                                                                                                                                                            -    - Fix errors and rebuild
                                                                                                                                                                                                                                                                                                                                             
                                                                                                                                                                                                                                                                                                                                                 - ### Run the Game
                                                                                                                                                                                                                                                                                                                                             
                                                                                                                                                                                                                                                                                                                                                 - 1. **Start the Game:**
                                                                                                                                                                                                                                                                                                                                                   2.    - Go to **Run → Run Project** (or press **F6**)
                                                                                                                                                                                                                                                                                                                                                         -    - Alternatively, right-click the project and select **"Run"**
                                                                                                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                                                                              - 2. **Expected Result:**
                                                                                                                                                                                                                                                                                                                                                                3.    - The game window opens (approximately 800x600 pixels)
                                                                                                                                                                                                                                                                                                                                                                      -    - You should see a road with cars and trees
                                                                                                                                                                                                                                                                                                                                                                           -    - The game starts immediately
                                                                                                                                                                                                                                                                                                                                                                            
                                                                                                                                                                                                                                                                                                                                                                                - 3. **Game Controls:**
                                                                                                                                                                                                                                                                                                                                                                                  4.    - **LEFT ARROW** - Move player car left
                                                                                                                                                                                                                                                                                                                                                                                        -    - **RIGHT ARROW** - Move player car right
                                                                                                                                                                                                                                                                                                                                                                                             -    - **SPACE BAR** - Restart game after crash
                                                                                                                                                                                                                                                                                                                                                                                                  -    - **Close window** - Exit game
                                                                                                                                                                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                                                                                                                                                                       - ### Debugging
                                                                                                                                                                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                                                                                                                                                                       - If the game doesn't run:
                                                                                                                                                                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                                                                                                                                                                       - 1. **Check Java Installation:**
                                                                                                                                                                                                                                                                                                                                                                                                         2.    - Go to **Tools → Options → Java**
                                                                                                                                                                                                                                                                                                                                                                                                               -    - Verify JDK is properly configured
                                                                                                                                                                                                                                                                                                                                                                                                                
                                                                                                                                                                                                                                                                                                                                                                                                                    - 2. **Check Project Properties:**
                                                                                                                                                                                                                                                                                                                                                                                                                      3.    - Right-click project → **Properties**
                                                                                                                                                                                                                                                                                                                                                                                                                            -    - Go to **Build → Compile**
                                                                                                                                                                                                                                                                                                                                                                                                                                 -    - Ensure Java Platform is "JDK" (not "JRE")
                                                                                                                                                                                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                                                                                                                                                                      - 3. **Check Image Files:**
                                                                                                                                                                                                                                                                                                                                                                                                                                        4.    - Verify all 4 image files exist in the `images` folder
                                                                                                                                                                                                                                                                                                                                                                                                                                              -    - Right-click **`images` folder** → **"Open in Explorer"**
                                                                                                                                                                                                                                                                                                                                                                                                                                                   -    - Confirm `car1.png`, `car2.png`, `car3.png`, `tree.png` are present

                                                                                                                                                                                                                                                                                                                                                                                                                                                        ## Troubleshooting
                                                                                                                                                                                                                                                                                                                                                                                                                                                        
                                                                                                                                                                                                                                                                                                                                                                                                                                                        ### Problem: "Cannot find image file" Error
                                                                                                                                                                                                                                                                                                                                                                                                                                                        
                                                                                                                                                                                                                                                                                                                                                                                                                                                        **Solution:**
                                                                                                                                                                                                                                                                                                                                                                                                                                                        1. Verify image files are in the `images/` folder (not a subfolder)
                                                                                                                                                                                                                                                                                                                                                                                                                                                        2. 2. Check filename spelling matches exactly:
                                                                                                                                                                                                                                                                                                                                                                                                                                                           3.    - `car1.png` (lowercase)
                                                                                                                                                                                                                                                                                                                                                                                                                                                                 -    - `car2.png` (lowercase)
                                                                                                                                                                                                                                                                                                                                                                                                                                                                      -    - `car3.png` (lowercase)
                                                                                                                                                                                                                                                                                                                                                                                                                                                                           -    - `tree.png` (lowercase)
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                - 3. Ensure images are PNG format, not JPG or other formats
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  4. 4. Right-click images folder in NetBeans → **"Refresh"**
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     5. ### Problem: "Compilation Error - Cannot Resolve Symbol"
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     6. **Solution:**
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     7. 1. Go to **Build → Clean and Build Project**
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        2. 2. Wait for rebuild to complete
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           3. 3. If errors persist, check that all Java files are present in CarProject folder
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              4. ### Problem: Game Window Appears but Nothing Shows
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              5. **Solution:**
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              6. 1. Verify image files are in the correct format (must be PNG)
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 2. 2. Check image dimensions aren't too large (recommend 40-60px width)
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    3. 3. Ensure images have proper transparency
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       4. 4. Rebuild project and restart game
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          5. ### Problem: Game Runs Slowly
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          6. **Solution:**
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          7. 1. Close other applications
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             2. 2. Check image file sizes (should be under 50KB each)
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                3. 3. Verify computer meets minimum requirements:
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   4.    - Windows: Java 8+, 512MB RAM
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         -    - Mac: Java 8+, 512MB RAM
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              -    - Linux: Java 8+, 512MB RAM
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   - ### Problem: "Build Failed" Message
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   - **Solution:**
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   - 1. Check the **Output** panel for specific error
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     2. 2. Most common causes:
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        3.    - Missing image files → Add images (see Step 4)
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              -    - Java not found → Install JDK
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   -    - NetBeans Java settings incorrect → Check Tools → Options
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        - 3. Right-click project → **"Properties"** → verify Java Platform
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          4. ### Problem: Cannot Find NetBeans Menu Options
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          5. **Solution:**

                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             2. 2. Alternative keyboard shortcuts:
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                3.    - **Build Project:** `Ctrl+Shift+B` (Windows/Linux) or `Cmd+Shift+B` (Mac)
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      -    - **Run Project:** `F6`
